# Roadmap

## Current public preview

- Provider-neutral AI gateway with Anthropic, OpenAI, and Gemini adapters.
- OpenAI-compatible `/v1/chat/completions` front door.
- Claude.ai browser extension controls.
- Best-effort hosted-web model rewrite/block support.
- Policy bundles, audit ingestion, event buffering, identity/device-trust scaffolding.
- Admin console and policy simulator.

## Near-term

- Demo mode with seeded dashboard data and mock provider responses.
- Streaming response support.
- Tool/function-call policy normalization.
- SIEM/S3 export pipeline.
- MCP server/tool governance.
- mTLS/MDM-issued device certificates.
- Azure OpenAI provider adapter.
- Terraform/ECS/EKS production deployment modules.

## Later

- Managed policy service.
- Advanced content classification/DLP integrations.
- Browser support expansion.
- Endpoint hardening and tamper detection.
- Formal security review.
